❤️ Heart Disease Prediction Web App

Project Overview

This project is a Machine Learning-based Heart Disease Prediction System that predicts whether a person is likely to have heart disease based on medical attributes.

The model is trained using a heart disease dataset from Kaggle and deployed as a web application using Flask. Users can enter medical details into a web form, and the system predicts whether the person has heart disease or not.


Technologies Used
🔹 Programming Language

Python 3.x

🔹 Python Libraries

Pandas – Data manipulation and preprocessing

NumPy – Numerical computations

Scikit-learn (sklearn) – Machine learning model training and evaluation

Pickle – Model serialization (saving and loading trained model)

Flask – Web framework for deploying the ML model

🔹 Frontend

HTML – Structure of web pages

CSS – Styling and design of web interface
